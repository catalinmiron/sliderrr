### Getting it up and running

1. Download the latest release and unzip
2. `cd popppular`
3. `node server.js &`
4. Hit `yourserveraddress:8080`

### Developing locally

1. `git clone git@github.com:iest/popppular.git`
2. `cd popppular`
3. `npm install`
4. `node server.js`
